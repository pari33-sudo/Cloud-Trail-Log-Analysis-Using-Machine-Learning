## **🆘 Troubleshooting**

### **"Command not found: git"**
**Solution:** Git isn't installed
- Download from: https://git-scm.com/download
- Install it
- Restart your terminal

### **"Permission denied" error**
**Solution:** Authentication issue
- See: `FIX_GITHUB_PERMISSION_ERROR.md` in outputs

### **"No such file or directory"**
**Solution:** Wrong folder location
- Run: `pwd` (Mac/Linux) or `cd` (Windows) to see current location
- Navigate to your cloud-trail-log-analysis-ml folder

### **Files not showing on GitHub after push**
**Solution:** Check .gitignore
- Make sure .gitignore isn't excluding your important files
- If stuck, try: `git add --force filename`
